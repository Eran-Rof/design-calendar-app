## How to update this doc

When a phase or module lands (or partially lands), in the SAME PR:
1. Flip its row status (вђЬвЖТрЯЯ°вЖТвЬЕ) + add the PR number(s) in Refs.
2. Re-count the Summary table (done phases / 25, done modules / 49) and update the three %s.
3. Bump **Last updated**.

Keep it honest вАФ рЯЯ° for "arch/partial," вЬЕ only when the phase is functionally shippable. The % is derived from the rows, so the rows are the source of truth.
